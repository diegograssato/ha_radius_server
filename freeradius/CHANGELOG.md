# Changelog

## 1.0.0

- Versão inicial do addon FreeRADIUS
- Suporte a EAP/PEAP, EAP/TTLS e EAP-TLS
- Geração automática de certificado auto-assinado
- Usuários locais configuráveis via addon options
- Integração opcional com SQLite, MySQL e PostgreSQL
- Suporte a multi-arquitetura: amd64, aarch64, armv7
- Validação de configuração antes de iniciar o serviço
